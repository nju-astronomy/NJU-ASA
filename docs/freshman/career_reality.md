# 兴趣之外：天文作为职业的另一面
>当“我这辈子只做天文”的最初热情慢慢可能因为成绩、压力等因素而被你重新考虑之后，如果将天文作为职业选择，这条路在时间、精力、生活方式和机会成本上意味着什么？

如果看到这里你对自己是否愿意做天文研究有了进一步的思考，那么恭喜你，你更好地认识了自己！如果你发现你还要再思考思考，那么，有句俗话说的好，爱上一个人很容易，一直爱下去往往很难。为什么网上会充斥着大量对于基础学科专业的劝退贴，可能因为情绪宣泄与个例放大有些夸张，不过一般来说，每一届进入天空院的同学中，未来真正长期从事天文科研的人，可能只有 20% 甚至更少。当然，这并不意味着本科阶段选择天文或物理是错误的决定，其实这样的专业训练依然具有很强的通用价值与竞争力。（见[天文之外：后来那些不做天文的人都去哪了？](../beyond_astro/leaving_astro.md)）

但在做出“是否将天文作为职业道路”这一决定之前，对这条路径本身有一个尽量清醒的认识，仍然是必要的。在现在的大环境下，如果你希望未来在天文学界获得一份教职，这条路径意味着：在本科阶段，你需要具备足够突出的能力，支撑你继续进入研究生阶段；若直接申请国外的 PhD，按照目前的竞争态势，你的综合能力很可能需要位于中国大陆天文相关本科生的前 1%。如果选择在国内体系中继续深造，那么在天空院，你大致需要处于前 50% 才具备较为稳妥的保研资格。进入研究生阶段后，科研能力、论文质量、研究方向与导师资源等因素将进一步叠加，你仍需处在同龄人中的前列，才有机会进入一个较好的博士后平台。而博士阶段通常需要 5-6 年，一段博后一般 2-3 年。若希望在南大层次的高校获得教职，现实中往往需要经历两期甚至更多博后，并且是在世界范围内较为顶尖的科研机构之间不断流动。这样的好处在于，可以边科研边环游世界，认识更多的人，与不同的人打交道，体验不同的文化，但一方面就是回过头来看，人生往往已至三十岁出头，居住地频繁更换，这条路是否是你认同的或者承担的，值得认真思考。

此外，对于科研，努力了不一定有回报，不努力肯定没有回报。方向选择、研究环境、机遇本身都会影响，可能在一个课题上投入多年，结果因为一些原因没有产出；也可能因为导师、合作者或外界的压力，遭遇与你能力并不完全匹配的结果，有能力却未必获得理想回报，在科研中并不罕见。因此，如何在科研之外，为自己构建相对稳定、可预期的正反馈来源，往往比想象中更重要。	

我们的本意绝对不是劝退！即便最终没有选择继续走在天文科研这条路上，这段经历本身，也是一段非常重要的积累。天文与物理训练教会我们的，远不止天文知识。它塑造的是你面对未知问题时的耐心、拆解复杂问题的能力、在长时间没有答案的情况下仍然推进工作的习惯，以及对“不确定性”的容忍度。这些能力，在科研之外的许多领域——无论是工业界、数据科学、金融、工程、教育，还是完全不同的人生选择中——都具有长期价值。但很多时候，“我喜欢天文”和“我适合以天文研究为主要职业”，并不是同一个问题。（Malin，天体物理方向，2022级本科生）
