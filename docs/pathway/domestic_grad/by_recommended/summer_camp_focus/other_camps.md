# 其他学校

## 紫金山天文台

紫台的夏令营报名比较简单，网上报名并提交申请表和证明材料就行，甚至不需要推荐信。夏令营的前面几天会有台内各个方向的老师来做报告，介绍各自领域的师资力量、研究规划等基本情况。此外还安排了去紫台老园区、天仪中心和紫台实验室参观的活动。面试前一天的下午是自由交流环节，会在紫台的报告厅举行。每个课题组都有对应的展位并安排有一两位老师在场，同学们可以根据自己的兴趣选择对应方向的老师面对面咨询交流。如果同学已经有了心仪的课题组或心仪的导师，那么这其实也是一个很好的机会，可以积极地向老师介绍自己的个人情况，表达个人意愿，增进老师对你的了解。因为紫台的面试是按专业分组进行的，而且和你交流的老师很可能会出席第二天的面试考核，所以提前联系联系还是挺有必要的。

在面试前一天会要求选择专业意向，有天体物理、天体测量和天体力学、天文技术与方法三个方向。之后会公布第二天的面试顺序并让同学把个人陈述PPT发到指定邮箱。面试的流程相对来说也挺简单的，就是3-5分钟个人陈述+提问回答+英语考核。个人陈述的的准备可以看欧阳老师写的内容（见上）。之后的提问环节老师会更侧重于你的科研经历，基础知识的考核占比不是很高（但也得好好准备）。老师的提问有一定的压力，会围绕着你的项目一直追问。当然也不必过于紧张，老师更多的是引导你对自己的项目展开新的思考。英语考核我所了解的有两种形式，第一种是有一位老师会用英语问你一个问题，然后让你用英语回答。问题不会很难，比如会让你介绍相关领域的某一个事物或现象，简单回答即可。第二种是给你一段短文阅读并翻译（之后可能会有提问？这个确实记不清了）。总体来讲紫台的的面试考核难度不是很大。（PS：紫台的优营通知比较阴间，会在9月份的时候才打电话告诉你，所以面试后不用过于着急等待结果。不过我感觉报紫台夏令营的好处是不用跑很远，毕竟就在学校旁边。。。）
（王彦恒，天力天测方向，2022级本科生）
