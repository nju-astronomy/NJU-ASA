## 程鑫联系项目：

> MPS 德国马普太阳系研究所 导师Pradeep Chitta

我们院的大部分老师都有海外合作。如果想要通过导师的联系申请海外的intern，最方便的方法就是在导师个人主页了解导师的海外经历（博士博后、访问学者等），参考导师论文作者栏中的海外合作者，然后和导师直接讨论intern意向。我们这个项目是程鑫老师联系的，他曾经在马普太阳系所（Max Planck Institute for Solar System Research, MPS, https://www.mps.mpg.de/en ）做了三年洪堡学者，后来至今长期和MPS的Pradeep Chitta、Hardi Peter等研究者合作。他几乎每年都会派研究生到MPS联培，有时也会把机会给本科生（比如我们这次），作为Visiting Student（=Research Intern）到MPS。

我们使用的是比较方便的申根签证，在申根区居留上限90天，项目时长因此定为大概小于等于3个月。这种方式比较轻松，只需要用学院或者学校开的资助证明、对方机构邀请信，以及其他要求的个人信息作为签证材料即可，且南京也有代理签证中心，一到两个月可以搞定签证，比较便捷（除了过程中VFS官网不好用和签证中心会想方设法赚额外费用之类的以外）。如果选择的intern是时长超过三个月、或者有APS要求，签证材料和APS审核会比较复杂，假如自己能选定时间和方式，或许可以参考自己是否有时间和精力来权衡。这些都可以查到很多攻略帖子。此外短期在MPS做intern无需学习德语，在马普所用英语，在外生活用英语+翻译软件即可，当然学习一些德语更便于交流。

MPS有从太阳/恒星内部到太阳大气、空间物理等方向比较全面的多个研究组，还有行星科学组；MPS且有不少观测仪器项目甚至系内行星深空探测项目，例如Solar Orbiter、Proba、JUICE等，也在Solar Orbiter的PHI、EUI等仪器高分辨率观测和相关模拟研究方面很出色。Pradeep本人非常nice，完全不push，也非常尊重学生意愿，讨论和指导很细致认真，细到文件格式是否有损、一起给代码debug等。他主要做太阳高分辨率观测，也参与做模拟，同时参与ESA一些探测器研发；近年来重点工作围绕小尺度喷流和爆发活动，并且有ERC Starting Grant，组内人不多（一个做模拟的博后，一个做观测的博士和一个硕士）。Hardi Peter已经比较老资历，算是MPS太阳大组的Leader，之前有研究生联培的指导老师是Hardi Peter，但现在不确定本科生是否还可以找上他。

如果对太阳物理感兴趣，可以考虑寻找机会到MPS做intern（如果考虑读博的话，那边的博士是和哥廷根大学或者TU Braunschweig一起培养），那边在太阳物理这个方向上还是强的，可以在太阳物理相当多文章的作者栏看到MPS。例如北大田教授、天空院陈枫老师等相当多国内太阳物理方向研究者都在MPS学习工作过，还有相当多像程鑫老师一样在MPS做过访问学者的。现在有至少十几位北大、中科大、南大的博后或者研究生正在MPS做研究，在那边intern也可以结识到很多太阳或相关方向的前辈，多多交流可以学习到很多，最主要是他们会带去吃当地好吃的（）。

MPS在哥廷根，一个“适合做学术”的小城，普朗克、高斯等人长眠在这里；市内商店比较齐全，满足日常生活没问题，但是哥廷根小城市周日大部分商店和超市不开，柏林、法兰克福这些大城市/旅游城市周日商店开门的比例大一些（各地亚超有不少周日开的）。如果有购买电器/衣服等需求可能需要去其他城市会更好，此外这里离周围大城市有点远。如果租住MPS的guesthouse（周围超市比较多，地方很方便），价格500～700欧一月，家具和每周清洁等包在内，其余生活费一般一月500到700欧左右，MPS或者哥廷根大学食堂一餐大约5～8欧。

此外，申请（联系）到交换或者intern之类项目之后务必向辅导员和教务员询问出国手续，尤其注意找学院负责海外交流的老师开资助证明（学院资助的情况下）、在学校OA系统上填出国（境）审批（很重要！）。（庞庆锋，王俊杰，天体物理方向，2022级本科生）

## 个人项目：

> USM/LMU 德国慕尼黑大学天文台 导师 Daniel Gruen

我这个其实算是自己套瓷的项目，简单来说就是我直接发邮件询问对方教授收不收学生到他那做一段时间实习，在邮件里你可以基本说清楚你的简历和研究经历，然后表达清楚自己为什么对对方的课题组感兴趣，我当初是因为想在申请前出国看一看情况，然后Daniel是做观测宇宙学的，对这个方向当初比较心动所以想多了解一些（现在我并不打算以后做这个方向哈哈哈哈），这个属于海投了，如果有条件的话，据我所知，我们学院有不少老师会有connection在本科期间提供给同学们出国实习的机会，这种也会方便一些。慕尼黑天文台这边对于访问学生的接受有些麻烦，对方同意的话会给你开具介绍信，然后你需要在LMU的申请系统里面提交材料申请“program student”，这个过程最麻烦的点在于你需要去考APS，APS（留德审核部）是一个审核你是否具备德国高校入学资格，同时确认您提交的各类中国大学文件是否真实（实际上我觉得就是对中国人的歧视）。这个考试你需要去上海进行面试然后考察你一系列的学业和语言水平是否过关，本来如果你是学校层面过去交换的学生是不需要这个的APS证书，但是奇葩的德国人对于“program student”这个项目也需要你考这个，否则你无法注册成为“program student”，当然这个也可能是Daniel要求我这样做（说是要保证我在德国的安全、保险一系列事情巴拉巴拉），如果问别的老师或许不需要那么麻烦。之后就是正常申请签证、准备的流程，多说一句，德国的签证也是有点材料需要的过于多了。

不过有机会的话，我会推荐大家来慕尼黑进行交换，除了上面说的你自己套瓷或是导师有关系推荐你，还有你可以作为交换生的方面去慕尼黑大学（LMU）或者慕尼黑工业大学（TUM）交换（这两所学校每年在交换生系统上都有名额开放，应该不是很难申请），这样你也有机会在那边和老师做项目。慕尼黑绝对是欧洲天文的中心之一，USM（慕尼黑天文台）、ESO（欧洲南方天文台）、MPE（马克斯·普朗克地外物理研究所）、MPA（马克斯·普朗克天体物理研究所）、MPP（马克斯·普朗克物理研究所）都在这里，上述研究所的含金量不必我多说，并且相互之间的合作、交流非常密切，每个所之间大家的报告、活动很多都是共同举办的，慕尼黑的理论物理、宇宙学、计算天体物理等等方向都非常强势。

日常生活层面（比如银行、租房合同、政府部门文件）会德语会更方便，但不是刚需。生活成本方面，慕尼黑确实是德国最贵的城市之一。合租公寓（WG）目前普遍在 700 欧以上，低于这个价格的房源要非常谨慎辨别，很多要么条件较差，要么存在诈骗风险。单人公寓通常会更高。饮食方面，如果自己做饭，一个月 350-450 欧左右基本可以覆盖日常生活开销；外出就餐则明显偏贵，个人觉得德国菜不好吃。德国超市一般晚上 8 点左右关门，周日基本不开门，这一点刚来时需要适应。娱乐活动相对不像柏林那么丰富，但城市整体安静有序，据他们讲慕尼黑和巴伐利亚地区偏老钱风哈哈哈哈。

治安方面，慕尼黑在德国乃至欧洲都属于非常安全的城市。个人生活体验中很少会遇到严重治安问题。当然，任何大城市都存在小偷等风险，基本的安全意识还是需要有。一个比较大的优势是德国的公共交通体系。购买 Deutschlandticket（月票）后，可以乘坐全德国范围内的区域公共交通（不包括 ICE 高铁），这对周末出行非常方便。慕尼黑位于德国南部、阿尔卑斯山脉北麓，周边有大量湖泊和山脉资源，非常适合周末去徒步、滑雪和户外运动。（欧阳马霖，天体物理方向，2022级本科生）

王瑜老师和柳若愚老师是研究生同学（connection在此xs），然后研究所在pescara，住在他们的guest house（所以不用自己找房！也不用付房租！水电网全包！就是房子不是很新，不过就住在美丽海滩边！问题不大！）。王瑜老师是ICRANet的所长，还有一位老教授 Remo Ruffini（80+高龄依旧每天工作aaa甚至周末两天都不休息，精力太旺盛了...），Ruffini 典型热情洋溢且脾气火爆老意大利人，但似乎对中国文化和中国人情有独钟所以对我们非常和蔼，他会想介入所有所里在开展的工作所以他也可以算导师之一吧...（不过请注意老人家确实思维已经不活络了也经常忘事或者会叫你做一些和你项目本身无关的事情，大家都哄着他顺着他嗯...主要是他老人家的同门好几个都得过诺贝尔奖了能看出他有点想再努力一把的）
