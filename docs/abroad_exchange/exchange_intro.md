> 本科出国交换项目对学业和科研有什么影响？申请有什么要求？

我们一般会以“交换”来代指所有的本科期间前往境外院校进行为期数月到一年不等的学习或实习的项目。它大致分为两种模式：

- 交换生（Exchange Student）：指的是前往境外高校进行一个学期的课程学习。主要内容是前往境外高校，编入对应的学院和当地学生一起上一学期课并获取相应学分。它是“交换”这个词的最原始含义。一般是由学校的国际处派出，属于校际学生交流项目。你在国际处官网上看到的“XX学期XX大学交换项目”基本都属于这一种。

- 科研实习（Research Internship）或者叫学术访问（Academic Visit）：指的是前往境外科研院所跟着当地导师开展一般为期数月的科研，具体参加时间比较灵活。以学院和导师个人项目为主。纯科研导向，一般无需进行强制的课程学习，也无学分和成果要求。常见的暑研（Summer Intern）也属于这一类别。

对于交换生项目，一般需要通过[国际处官方渠道报名](https://stuex.nju.edu.cn/ )，和全校范围内的申请者一起竞争名额。
      
申请要求方面，基本上所有的项目都会要求你具有相应的语言水平，例如托福或者雅思。一些项目还会要求申请者的学分绩必须高于某一个门槛。具备基础的资格后，会由国际处进行筛选择优录取，一般没有面试只看提交的资料。除了美国院校以外，一般没有学费，但生活费需自理。参加该类项目所产生的费用原则上不可报销。

由于参加交换生一般会是一整学期在外派出，且派出学期不得在南大选课，所以需要提前规划课程学习，将一些专业准出课程先修或者缓修，否则容易影响毕业，得不偿失。或者也可以在交换单位学习相关课程，返校后再做学分转换。但多数情况下境外院校开设的课程并不满足南大的课程标准，难以转换为必修课学分。此外转换学分的手续也比较复杂。因此要谨慎对待。

对于科研实习项目，一般是自己联系或者走学院或导师的项目。学校层面也有郑钢海外奖学金这种项目。

由于面向的对象更少，所以一般竞争很小或者没有。基本上不会对语言成绩等有硬性的要求，标准比较灵活。一般学院会组织老师先进行面试，主要是看你的目的规划和基本的语言水平和成绩，并不会特别困难。出发之前也会需要和境外导师线上沟通，但基本也不构成筛选。近几年学院的项目基本上只要报名，通过率都比较高。对于这类科研项目，一般也只需要出生活费即可，且学院可以给予一定的报销。

参加此类项目会遇到和交换生项目类似的问题，即课程的学习安排。一般也需要将交流学期的必修课程调整到其它学期。不过此类项目的优势在于，你可以选择在大四修完所有课后再派出交流，或者如果交流时间较短的话可以选择将一些课程免修不免考，学院的老师一般都会理解的。

从内容上来看就容易发现，这两种项目有着非常不同的导向。对于希望之后申请国外院校的同学来讲，由于申请中非常看重connection和publication，因此多数时候科研实习会是更合适的选择。它能够让你提前体验“在国外进行科研是一件怎样的事”，以及能帮你建立一些宝贵的conncection（直接体现就是推荐信）。如果能做出一些成果写成paper，那更是锦上添花，会大大增加你的申请资本。因此，在下面的部分中，我们会将介绍的重心放在科研实习项目上。而交换生项目，由于只是课程学习，所以一般难以建立很强的connection。更多人选择交换生项目只不过是希望能在本科阶段出国体验一下不同的文化，对于国外院校申请其实帮助有限。当然，也不排除一些特殊情况的存在。例如，你正好有强烈的意愿想要申请交换生项目列表中的某所学校，且非常确定通过交换生项目就可以获取到你所需要的申请便利，为你的申请谋取较大的增益；或者在某些导师比较难以为你提供签证的地区，通过交换生项目获取当地的签证，然后利用交换机会在当地实际进行科研实习也未尝不是一种曲线救国的方法。但总体上来讲，选择有针对性的科研实习或者学术访问对于出国申请的增益往往是更大的。当然，如果你只是想用一个学期去国外体验一下，那也未尝不可。总之，还是那句话，选择适合自己的就是最好的。（汪永正，天体物理方向，2022级本科生）
