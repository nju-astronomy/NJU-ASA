## 项目介绍和申请经验

郑钢海外奖学金项目由南大杰出校友郑钢学长设立并长期支持，旨在资助学院优秀本科生赴美国高校或科研机构开展科研实习。该项目每年资助金额为每人3万元人民币，时间一般安排在大三暑假或大四上学期，持续三个月至半年不等，具体时长通常取决于签证办理进度。

天文方向通常会对接一般3到4位合作导师，每位导师选拔1名学生。选拔流程由学院统一组织报名，并通过答辩形式进行评审。竞争情况因年份而异，有些年份报名人数较少，竞争相对缓和；也有些年份某位导师名下会有多名同学报名，需要进行更充分的展示与比较。项目本身没有硬性语言成绩提交要求，更注重申请者的科研潜力、专业基础以及与导师方向的匹配度。（欧阳马霖，天体物理方向，2022级本科生）

## 导师介绍	

Greg 和 Mark 两位导师的实习机会，都与[清华大学李辉老师](https://astro.tsinghua.edu.cn/info/1040/1048.htm)的推荐相关。李辉老师本科和硕士就读于南大，博士毕业于密歇根大学，随后在 MIT 和 Columbia 从事博士后研究，所以在具体实习对接过程中，主要由李辉老师协助与 Greg 和 Mark 沟通实习安排及相关事宜。（欧阳马霖，天体物理方向，2022级本科生）

### Greg Bryan（Columbia）

Greg是Columbia天文系的前系主任，做大型辐射流体宇宙学模拟和理论，对星系形成、宇宙学模拟、早期宇宙等多个方向都有所涉猎，他代表性的工作有早期宇宙第一批恒星的形成、X-ray cluster等，他也是Enzo代码（如果你做计算天体物理应该听说过）的主要开发者之一。Greg的研究兴趣很广泛，近些年他领导了一个大的合作组织Simons Collaboration on Learning the Universe，该计划尝试结合宇宙学模拟、高精度巡天数据和机器学习，试图反推宇宙的初始条件并建模更精确的模型，也包括对下一代宇宙学模拟sub-grid model的开发，组织里汇集了CCA、Princeton、Harvard、MPA等全球多家和计算天体物理相关的教授、学生，具体信息参见https://learning-the-universe.org/ 。Greg 在其中主要研究下一代宇宙学模拟子模型（Sub-grid models）的开发，研究星系形成、星系周介质（CGM）及黑洞（BH）等等对象的物理怎么协同宇宙演化，除此之外他也在关注对机器学习/人工智能在这方面的运用（他手底下项目感觉非常多）。

虽然感觉他平时很忙，但Greg对待学生很亲力亲为，为人很和善，也不怎么push，我去交换的时候每周会和我有两次单独讨论，外加一次小组的组会，除此之外我会去旁听他在学校开设的 physical cosmology和 statistics&numerical method（秋季学期）。Columbia在纽约市，纽约相信大家都有所耳闻，如果有机会来到这个生活一段时间是一段非常难忘的经历，你可以找到任何你想玩的和想吃的东西，在纽约教会我除了学术上的非常多的经验和想法，也教会我怎么平衡好生活和学习，在这样一个快节奏也高能量的地方生活对我来说是一件很幸福的事情，不过不一定适合所有人，纽约的治安不同地区不太一样，比如北边100街往上黑人居住的哈莱姆区以及布鲁克林南边我就从来都没有去过，大部分地方是比较正常的，除了街道脏、随时随地的流浪汉以及一直在耳边环绕的警笛声、时不时闻到的大麻味，当然如果你住在哥大附近、或者上东区、LIC、midtown会好一些（xhs上有很多租房攻略），以及随之而来的高昂的生活成本。纽约的房租，合租起码需要1800刀以上，生活费对我来说平均一个月1200刀，这是一周我可能会出去吃饭2到3次（平均一顿30刀以上 好一点50刀）然后其余时间在家做饭，然后有时间可能去看看百老汇，外面玩一玩（对不起纽约好玩的实在是太多了）的水平，如果节省一点，一个月800刀左右可能可以，取决于自身。需要提醒的是纽约人很多，人的素质层次不齐，租房就是你第一个要谨慎的事情，xhs上泛滥着非常多的租房中介，不要去找他们（大部分都非常不靠谱），可以在xhs上找学生的短租这个最方便，或者在正规的租房平台上找房，最保险的方法是，到了纽约后线下去看房。总之奖学金的 3 万人民币补贴在纽约肯定是不够的，尤其是房租占比很高，因此一定要提前做好财务规划。（欧阳马霖，天体物理方向，2022级本科生）
