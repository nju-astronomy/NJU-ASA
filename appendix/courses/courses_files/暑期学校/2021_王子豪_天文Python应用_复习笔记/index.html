    IV. AstroPy基本使用

    1. 天文数据类型

    FITS文件：普适图像传输系统

    结构：HDU（header and data unit）

    头结构（header）：包含多个关键字即信息

    第一行：SIMPLE=T

    第二行：每一个像元用多少字节表示 BITPIX

    第三行：数据轴（n个轴） NAXIS

    第四行：NAXISn

    $N_{bits}=|BITPIX|\times(NAXIS_1\times...\times NAXIS_n)$

    例如

    2维，每个轴14400个pixel，每个pixel占16bits

    时间格式：CCYY-MM-DD[Thh:mm:ss]
